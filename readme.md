## Music Player and Visualizer
Music Visualizer written in C for educational purposes using Raylib.

### Tested File Types
- MP3
- FLAC

### Hotkeys
`<Space>` Pause/Play  
`<G>` Show graph instead of bars  
`<R>` Replay dropped music  

#### Libraries used:
- [Raylib](https://www.raylib.com/)
